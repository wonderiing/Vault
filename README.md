# Base de Conocimientos 

Sitio web personal donde comparto mis notas de **desarrollo backend** y **ciberseguridad**, creando una base de datos organizada de mis conocimientos y aprendizajes. Por ahora solo se encuentra con notas de CTFs.

## Propósito

Este proyecto tiene como proposito replicar mi vault de Obsidian donde tengo notas de CTFs, desarrollo backend, cloud.

- **Desarrollo Backend**: Notas sobre Java, NestJs, Spring Framework, arquitecturas, patrones de diseño etc etc etc
- **Ciberseguridad**: Write-ups de CTFs, técnicas de pentesting, herramientas de hacking ético
- **Cloud & DevOps**: AWS, Docker, y otras tecnologías relacionadas
- **Frontend**: VUE te amo vue eres lo mejor.
