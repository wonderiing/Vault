# Base de Conocimientos

Colección personal de notas sobre distintos temas que he estudiado y quiero seguir organizando, con mayor enfoque en backend y cloud. También incluye ciberseguridad desde un lado más recreativo, principalmente con CTFs, y próximamente más contenido sobre CI/CD, DevOps y otras áreas que vaya aprendiendo.

La idea es mantener este espacio como una versión web de mi vault de Obsidian: simple, ordenado y en constante crecimiento.
