## **Grupos de Seguridad y redes:**

Los grupos de seguridad son reglas que tiene la instancia acerca de la entrada y salida de trafico

Allow SSH Traffic from: Permitir el trafico de SSH desde cualquier ip o desde la tu propia IP o una custom
Allow HTTPS/HTTP: Esto lo hacemos para permitir el trafico a traves de estos dos protocolos
