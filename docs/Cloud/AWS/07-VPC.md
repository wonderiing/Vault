
Una VPC es una red virtual que se puede usar para isolar recursos de AWS y tener todo en una misma red para mejora la latencia

- Una VPC puede contener subredes publicas y privadas
- InternetGateway se utiliza en las subredes publicas para la conexion a internet
- Las tablas de rutas son como un **mapa de caminos que la red usa para decidir por dónde enviar el tráfico** para llegar a un destino específico
