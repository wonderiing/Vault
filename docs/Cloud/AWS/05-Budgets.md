Un presupuesto nos sirve para que se nos notifique cuando cierto umbral de gastos que nosotros hayamos indicado sea excedido.

Para crear un presupuestos nos vamos al servicio de `Budgets`

![](assets/Pasted%20image%2020260709124836.webp)

En los detalles del presupuesto podemos configurar varias cosas como:

- El limite del presupuesto
- Servicios a los que se le aplica el presupuesto
- Periodo del presupuesto (Mensual, Trimestral, Anual o Personalizado)

![](assets/Pasted%20image%2020260709124957.webp)

Después deberemos crear una alerta en base a un umbral:

- En este caso, cuando se consuma el 50% de los 10 usd que establecimos como presupuesto, se nos mandara una alerta al correo electrónico.
