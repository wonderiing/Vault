## **Politicas y Ejemplos de Politicas**

El tema de politicas es absurdamente amplio y esta explicacion puede quedar muy vaga

En la pestana de permisos del bucket puedes editar las politicas
![](assets/Pasted%20image%2020250308154936.webp)


Puedes ver muchos ejemplos de politicas en la propia documentacion:
https://docs.aws.amazon.com/AmazonS3/latest/userguide/example-bucket-policies.html?icmpid=docs_amazons3_console

En este caso pusimos una politica de solo lectura, lo unico que tienes que cambiar es el `Resource` por el Bucket ARN, en el resource indicamos que afecta a todo el bucket usando el /* pero puede hacer que solo afecte a alguna carpeta 

ChatGPT te puede ayudar en el tema de politicas solo necesitarias pasarle el Bucket ARN
