IaaS:
- La infraestructura como servicio provee de recursos de computo al cliente, externaliza la configuración de servidores. Ej: AWS, AZURE, GCP.

PaaS:
- No se gestiona infraestructura pero se proporciona la plataforma para el despliegue de servicios. Ej: Vercel, Railway

SaaS:
- Proporciona el producto completo al cliente, no gestionas ni configuras nada solo utilizas el producto. Ej: Claude Code, Netflix, etc.
